坚持 毅力 信念

  • feeling
我想人大概还是需要坚持一件事情的,要学的有很多,不要只是为了开阔眼界去去学它,更应该去深入...去探究...

目前想要搞的东西有很多:
树莓派
gitlab
docker
vue
webapck 
react 
Nodejs
淘宝店
滑板
读书
写作
做饭
浏览器脚本
健身
羽毛球

这些东西 都是我的兴趣之一 我也都想好好培养这些兴趣,或做到一定的深度,可总是坚持不下去,我应该好好的计划一下,到底应该怎么做,坚持下去,做出一定的成果.是该好好的计划一下了,不能被外界打扰的培养自己的兴趣.

最近突然想到了创业这个事,对于我这样一个技术宅去想这种事总是会有点不切实际,我比较懒,又比较烦躁,怯弱,怕失败.这些缺点不克服,创业就是一件很难的事,是创业成就我 还是我成就创业那?这是一个问题.一个值得思考的问题

思考一下这些兴趣,有些是因为工作需要,有些是业余爱好,需要分出一个轻重缓急,一个优先级.保持专注,培养深度兴趣,体验乐趣.
